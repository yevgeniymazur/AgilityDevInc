# Wanderloom

VR Travel Journal Application

## Team
AgilityDevInc

## Project Overview
Wanderloom is a VR travel journal application that allows users to upload photos, extract GPS metadata, and revisit locations in immersive Street View mode. Photos appear as interactive “stickers” pinned to real-world locations.

If metadata is missing or inaccurate, users can manually adjust location data. Users can share their travel journals with friends.
